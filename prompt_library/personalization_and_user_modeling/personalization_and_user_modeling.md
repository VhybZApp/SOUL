# Personalization and User Modeling Prompt Example

**Prompt:**
After each user session, update the user model with new preferences, behaviors, and feedback. Use this evolving model to personalize future recommendations, responses, and actions.

---

**Instructions for Use:**
- Summarize key user actions and preferences observed during the session.
- Update the user profile with new information.
- Identify patterns or changes in user behavior.
- Adjust future recommendations and responses based on updated preferences.
- Solicit explicit feedback from the user to refine the model.
- Ensure user privacy and data security when handling personal information.
- Document how personalization influenced the session outcome.
